# Open Source Underwater Glider
## Underwater Glider
Underwater gliders are autonomous underwater vehicles used to collect valuable scientific data from water column with minimal effort. Since there is no external moving part, they are extremely efficient and can cruise the ocean for many months at a time.
![alt text](https://github.com/e-abdi/Open-Source-Glider/blob/main/Pics/Glider%20Example.PNG)
## This Project
This project serves as an effort in making underwater gliders more accessible to students and general public and to promote citizen science.
